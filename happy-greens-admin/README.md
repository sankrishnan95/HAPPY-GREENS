# Happy Greens Admin Dashboard

React + Vite + Tailwind admin panel for Happy Greens grocery store.

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Login

- Email: admin@happygreens.com
- Password: admin123

## Features

- Dashboard with stats
- Orders management with invoice printing
- Products & inventory tracking
- Reports & analytics with charts

## Backend

Connects to backend at http://localhost:3000
